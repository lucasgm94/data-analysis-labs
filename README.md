# Data Analytics & Business Intelligence Portfolio

This repository is a collection of data analysis projects focused on transforming raw data into actionable insights. It serves as a personal laboratory for exploring data cleaning, modeling, and visualization techniques.

---

## 🛠 Project Overview
The main goal of this space is to showcase end-to-end analytical solutions, with a strong focus on:
* **Data Transformation:** ETL processes and data cleaning.
* **Business Intelligence:** Interactive dashboards and storytelling.
* **Technical Stack:** Power BI, DAX, and SQL.

## 📂 Repository Structure
* **/power-bi-projects**: Contains `.pbix` files, datasets, and documentation for individual dashboards.
* **README.md**: Overview of the laboratory.

## 📈 Tools & Technologies
* **Power BI Desktop** (Data Visualization & Modeling)
* **Power Query** (M Language for ETL)
* **Microsoft Excel** (Data Sourcing)
